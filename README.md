# Softroids AI — landing page

Lekka, statyczna strona (jeden plik HTML, zero zależności) pod kampanie Google Ads.

## Pliki
- `index.html` — strona główna
- `polityka-prywatnosci.html` — polityka prywatności (RODO)
- `.nojekyll` — wyłącza przetwarzanie Jekyll na GitHub Pages

## Do uzupełnienia (placeholdery w kodzie `index.html`)
- **Web3Forms** — `WEB3FORMS_KEY` (klucz z web3forms.com dla hello@softroids.com)
- **Cookies / śledzenie** — `COOKIE_CONFIG`: `ga4` (G-…), `googleAds` (AW-…), `metaPixel` (ID)
- **KRS** — w `polityka-prywatnosci.html`, sekcja 1
- **canonical / og:url** — docelowy adres strony po podpięciu domeny

## Publikacja
Hostowane na GitHub Pages (branch `main`, katalog `/root`).

## Własna domena
Dodaj plik `CNAME` z domeną (np. `ai.softroids.com`) i skonfiguruj DNS u rejestratora.
